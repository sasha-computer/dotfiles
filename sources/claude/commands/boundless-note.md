# Save Boundless Note

Save a concise research note to the Boundless vault. Input: $ARGUMENTS

## Instructions

1. Parse the user's input or use recent conversation context to extract:
   - **Topic**: Create a descriptive title for the note
   - **Key findings**: The essential information to capture

2. Create a markdown file at `~/Documents/2026/Boundless/<Topic>.md` with:
   - A clear H1 title
   - A brief "Summary" section (1-2 sentences max)
   - Organized sections with bullet points
   - Reference links if applicable

3. Keep it concise:
   - No filler text or verbose explanations
   - Bullet points over paragraphs
   - Only essential information

4. Confirm the file path to the user.
